# Tax Form PDFs

Place official fillable PDF forms here. The form-filler.js will fill AcroForm fields in these PDFs.
If a form is missing, the system falls back to generating a summary PDF.

## Expected files:

- `f1040-2024.pdf` — IRS Form 1040 (2024)
- `f1040-2025.pdf` — IRS Form 1040 (2025)
- `ca540-2024.pdf` — CA FTB Form 540 (2024)
- `ca540-2025.pdf` — CA FTB Form 540 (2025)
- `ny-it201-2024.pdf` — NY DTF Form IT-201 (2024)
- `ny-it201-2025.pdf` — NY DTF Form IT-201 (2025)

## Download sources:

- IRS: https://www.irs.gov/forms-pubs
- CA FTB: https://www.ftb.ca.gov/forms/
- NY DTF: https://www.tax.ny.gov/forms/
