Delivery
Software Delivery Model
Ship new capabilities without firmware updates. Skills are markdown files, system updates use A/B partitions, and the device never depends on a cloud service to function.

Skills as Markdown

New capabilities ship as markdown files via Git. A lightweight agent on-device checks a manifest and pulls updates — no firmware flash, no downtime, no risk of bricking.

Git Markdown Manifest

Lightweight Update Agent

Agent on the device periodically checks the skill manifest, pulls new or updated markdown files, and makes them available to the LLM. Zero-touch deployment for new capabilities.

On-Device Agent Manifest Check Auto-Pull

OTA System Updates

Mender handles full system updates via A/B partition switching. If an update fails, the device automatically rolls back to the previous working partition. Industry-standard reliability.

Mender A/B Partitions Rollback

No Cloud Dependency

Core function runs entirely on-device. No cloud API calls for inference, no subscription required. The device works offline, updates are pulled when connectivity is available.

Model
Pay Once, Value Forever
No subscription model. Hardware margin funds development. Viable because inference runs locally — there are no per-query cloud API costs to recoup.

Local Inference = No Cloud Costs

The entire business model is viable because inference runs on the Jetson GPU. No cloud API calls means no per-query costs, no usage-based pricing pressure, and no reason to charge a subscription.

Hardware Margin Funds Development

2x BOM pricing gives healthy margin to fund ongoing software development. At Phase 2 pricing ($418–670 BOM, $799 retail), that's $129–381 per unit toward R&D, support, and new skills.

Hardware Generations Drive Revenue

New NVIDIA Jetson generations every 18–24 months create natural upgrade cycles. Each generation brings more compute, better efficiency, and new capabilities that drive repeat purchases.

Optional Premium Revenue

Two optional revenue streams for users who want more: a premium skills marketplace for specialized capabilities, and a cloud sync subscription for multi-device coordination.

Decisions
Key Decisions
The strategic choices that shape the manufacturing and scale-up roadmap.

Start With Dev Kit

No custom hardware risk at prototype stage. The Jetson Orin Nano Super Dev Kit ($249) is a production-ready board with all the compute we need. Custom carrier boards are a Phase 3 optimization, not a Phase 1 requirement.

Mender for OTA

Industry-standard A/B partition updates with automatic rollback. Open-source Community Edition for early phases, Enterprise for fleet management at scale. No proprietary lock-in, battle-tested on millions of embedded Linux devices.

Pay Once

No subscription model. Hardware margin funds development. This is only possible because inference runs locally — there are no cloud API costs that would force recurring revenue. Users own their device, period.

Skills as Markdown

Ship new capabilities without firmware updates. Markdown skill files are pulled via Git, interpreted by the on-device LLM. New skills deploy in minutes, not hours. No risk of bricking devices with bad firmware.

Domestic Assembly

Keep supply chain simple and local through Phase 3. Domestic contract manufacturers mean shorter lead times, easier quality control, and no international shipping complexity. Offshore only when volume justifies it.
